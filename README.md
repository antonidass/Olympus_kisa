# Кисы Олимпа — пайплайн генерации видео

Полный пайплайн создания коротких анимированных роликов по греческим мифам для канала «Кисы Олимпа»: сценарий → промпты → картинки → видео → озвучка → монтаж.

## Структура проекта

```
BOGI AI/
├── content/                          # Контент по роликам (один ролик = одна папка)
│   ├── Икар и Дедал/                 # Имя папки = название мифа с заглавных букв и пробелами
│   │   ├── prompts/                  # Промпты для генерации
│   │   │   ├── images.md             #   картинки (Google Flow / Nano Banana)
│   │   │   ├── video.md              #   видео (Veo / LTX)
│   │   │   └── voiceover.md          #   текст озвучки
│   │   ├── voiceover/
│   │   │   └── audio/                # Сгенерированные mp3 (не в git)
│   │   ├── images/                   # Сгенерированные картинки (не в git)
│   │   │   └── scene_01/             #   по одной папке на сцену, 1-4 варианта v1.jpg … v4.jpg
│   │   ├── video/                    # Сгенерированные клипы (не в git)
│   │   ├── music/                    # Фоновая музыка (не в git)
│   │   └── final/                    # Готовый результат (не в git)
│   │       └── Икар и Дедал.mp4      #   Имя файла = имя папки
│   ├── Сизифов Труд/                 # Новый сценарий — та же структура
│   └── Ящик Пандоры/
├── automation/                       # Скрипты автоматизации
│   ├── imagefx_runner.py             #   Playwright-раннер для Google Flow (картинки)
│   ├── video_runner.py               #   Playwright-раннер для Google Flow (видео Veo)
│   ├── elevenlabs_runner.py          #   TTS через ElevenLabs
│   ├── flow_projects.py              #   Общий резолвер Flow-проектов (ID хранится в json)
│   ├── flow_projects.json            #   Маппинг «имя сценария → flow_id» (обновляется при первом запуске)
│   ├── dump_media.py                 #   Дамп медиафайлов
│   ├── inspect_flow.py               #   Инспекция пайплайна
│   └── requirements.txt
├── remotion/                         # Remotion-проект (монтаж видео в React)
│   └── src/
│       ├── IcarusVideo.tsx           #   Главная композиция «Икар и Дедал»
│       ├── Intro.tsx                 #   Заставка «Миф за минуту»
│       ├── TextOverlay.tsx           #   Компонент текстовых субтитров
│       └── transitions/              #   Кастомные переходы между сценами
└── CONTEXT.md                        # Контекст проекта (правила, стиль, персонажи)
```

### Именование папок сценариев

Папка сценария внутри `content/` называется точно как миф: **с заглавных букв, слова через пробел, без подчёркиваний и транслита**. То же имя используется для финального видеофайла.

- ✅ `content/Икар и Дедал/final/Икар и Дедал.mp4`
- ✅ `content/Сизифов Труд/final/Сизифов Труд.mp4`
- ❌ `content/икар_и_дедал/` (старый формат, больше не используем)

## Установка

```bash
# Автоматизация (Python)
cd "BOGI AI"
pip install -r automation/requirements.txt
python -m playwright install chromium

# Remotion (Node.js)
cd remotion
npm install
```

## Пайплайн создания ролика

1. Предложить 5 сценариев. Я выбираю один.
2. Далее пишется сценарий по выбранному мифу и вместе с ударениями отправляется мне на проверку.
3. После правок он попадает в `content/<Название мифа>/prompts/voiceover.md`
4. Далее пишутся промпты картинок (`prompts/images.md`) и видео (`prompts/video.md`)
5. Озвучивается сценарий скриптом `automation/elevenlabs_runner.py`
6. **Картинки** — `automation/imagefx_runner.py content/<Название мифа>/prompts/images.md` (при первом запуске спросит Flow-project ID и сохранит его в `flow_projects.json`). Результат — `content/<Название мифа>/images/scene_NN/vN.jpg`
7. **Видео** — `automation/video_runner.py content/<Название мифа>/prompts/video.md`. Результат — `content/<Название мифа>/video/`
8. **Монтаж** — Remotion собирает всё в финальное видео с субтитрами, музыкой и переходами. Финал — `content/<Название мифа>/final/<Название мифа>.mp4`

## Правила контента

- Персонажи — антропоморфные бипедальные коты (не люди, не реалистичные кошки)
- Стиль — pixel art / 16-bit retro game style
- Ударения в текстах только на именах собственных (Ика́р, Деда́л)
- Видео-промпты обязательно содержат негатив: "No speech, no dialogue, no talking, no voices, no mouth movement, no music"
